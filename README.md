# M² - Personal Web Portfolio 🚀

Benvenuto nella repository del sito web di Matteo Medici, qui salverò i miei progetti e i miei contatti.

## 📝 Descrizione del progetto
Il sito è un portfolio minimale a pagina singola. Progettato per essere leggero, veloce e accessibile a tutti.

## 🛠️ Tecnologie Utilizzate
* **HTML5:** Per la struttura semantica della pagina.
* **CSS3:** Per la formattazione, i colori e il layout.

## 📁 Struttura dei File
* `index.html`: La pagina principale del sito.
* `style.css`: Il foglio di stile contenente tutte le regole grafiche.
* `/img`: Cartella contenente le immagini e le icone utilizzate.
